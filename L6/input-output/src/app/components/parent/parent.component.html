Child to Parent

If you recall the @Output annotation used in EventEmitters, it may also be used to send data from a child component to the parent. An event emitted by the EventEmitter can have some data attached to it, which can then be received by the parent component's handler for the custom event. Essentially, data can be passed from child to parent using event binding and passing data via an event.
